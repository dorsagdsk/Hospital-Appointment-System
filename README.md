# Hospital-Appointment-System

Java OOP project for managing hospital appointments.

Features:
- Patient registration and login
- Doctor registration and login
- Appointment booking and cancellation
- Daily visit capacity management
- Prescription creation
- Doctor specialties using Enum
- Object-oriented design using inheritance and associations

Technologies:
- Java
- Object-Oriented Programming (OOP)

Entities:
- Person
- Patient
- Doctor
- Appointment
- Prescription
